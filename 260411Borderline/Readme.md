再次感谢[@霧雨ResonantPsyche](https://space.bilibili.com/205919268)老师帮忙混音，又耐心听取我的意见修改了几稿；
这是我第一次挑战做大合唱，惊人地分出了8轨主声4轨和声，老师还是手到擒来。
这次也特别邀请了[@黑理miu](https://space.bilibili.com/2501388)老师帮忙绘制了封面，很好地还原出原封面的惊艳的感觉，细节把控和上色也十分优秀。

继Dye The Sky后，闪耀色彩又一首打动了我的曲子，果然还是全体曲合唱有气势。
作为最后一个加入闪耀色彩的成员，郁田阳希所属的小组黑色彗星(CoMETIK)其他两位成员都有断层高的人气，前有扭曲深残的神大人，后有出圈腹黑冈山妹；
与本作的追加组合中的一众问题儿童相比，小羊却一副软萌人畜无害的样子，但与樱木真乃那种略微电波而又天真，可远观而不可亵玩的气场不同，小羊显得更像一个常识人也更活泼和平易近人，一跃成为我在闪彩中最喜欢的角色。
在2025年的“螺旋 -Halo around-”公演上，Borderline第一次公开亮相，当年的年度视觉图和游戏封面，都是Borderline印象曲绘，穿着“反抗军”衣装的小羊站在中间位置，APP也图标也换成了小羊；而最近的闪彩官方打散重组的组合中，将小羊和老大鸽分到了一起，虽然是神必的昭和风，但果然小羊你就是闪耀色彩C位的接班人，小羊可以称帝了！
