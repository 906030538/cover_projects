如果有人要问为什么不含AVER？那自然是因为立项时AVER还没发售（
拖了那么久然后理所应当地撞车了，不过似乎很多人都没听过全曲，明明最精华的就是四重唱这句。
又调一首歌词带Hop step, Jump step的曲子。

LLLL坐拥反田叶月、进藤天音、渡濑结月、深川瑠华几个二偶新生代顶梁柱，制作人担当为BanG_Dream!原案的中村航，动画第二季也给主了戏份，剧情王道音乐好听，制作也很优秀；奈何国内EDM受众并不多，加上一些破事，D4DJ在国内整体都不愠不火。

第一次调EDM类型的曲子，虽然没有上一稿十多轨那么恐怖，也是主和各4轨，不过学习了很多调式的知识后，和声意外地不难扒。
这次也感谢霧雨ResonantPsyche老师帮忙混音，因为是EDM，调教实际上不需要太多技巧，反而是混音阶段需要处理得更多，也是反复修改了好几稿。
至于视频就立绘平移，甚至没有平移（
