## Comment

节后一个多月几乎没有休息，于是拖了这么久才投，而且还是半成品（逃

发售前就有这个想法，邦邦官方和啤酒烧烤联动的概率不大，那就得靠二创发力啦，也有有同样想法的人交流过，有种占坑的罪恶感。话说回来，ProjectSEKAI我也只是稍微了解过没有入坑，主要是没时间，平时主要关注声优梗(?)和新歌；虽然我向来喜欢开朗励志的角色剧情，但25时的风格意外戳我。
